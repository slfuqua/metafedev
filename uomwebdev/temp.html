Input Validation

JavaScript can be used for validating form inputs. Without JS, the form is only checked for validity when the submit button is pressed. We can use JS to actually check the input as soon as the user exits the input box. That way they can make corrections as they go.

The code below uses the method reportValidity() to trigger the browser's built-in form validation UI for an element or full form. We are going to use JavaScript's blur event, which occurs when an element loses focus, i.e., when the user moves to another element. We have set the text input's type to "email", and have made it required.
