Finding HTML Elements by Class Name

The getElementsByClass() method is used to retrieve all members of a class, or classes. Note that 'Elements' is plural.

We can retrieve all elements with one or multiple class names. And since it returns an array, we can also access its members using indexing. When using mulitple classes as arguments, getElementsByClass(), only returns those elements that have both (or all) of the elements.

In this example we will first change the styling of all three of these first three paragraphs with the class="ex4" AND class="ex5" selectors. And we will change the background color of the middle paragraph with indexing.

It will not work on this paragraph, because it only has a class of ex5, which is listed second in the method.
