Find an HTML Element by Class

The getElementsByClass() method is used to retrieve all members of a class, or classes. Note that 'Elements' is plural.

We can retrieve all elements with one or multiple class names. And since it returns an array, we can also access its members using indexing. When using mulitple classes as arguments, getElementsByClass(), only returns those elements that have all of the elements listed.

Using document.write()

document.write() is generally discouraged due to its potential to overwrite the entire document and its synchronous nature, which can block rendering. To modify an element, innerHTML can be used instead.

JavaScript Form Validation

Input Validation

HTML5 has a number of built-in validation attributes for form inputs, such as pattern for phone numbers or passwords, min and max for numbers or amounts, required, and input type for email, url, number, etc. But there are many checks that HTML can't complete, like checking a server for available usernames, making sure that a confirmed password matches the original, giving custom feedback on password strength or any other input, or triggering validation checks when exiting an input instead of waiting for the submit button to be clicked.

Without JS, the form is only checked for validity when the submit button is pressed. We can use JS to actually check the input as soon as the user exits the input box. That way they can make corrections as they go.
